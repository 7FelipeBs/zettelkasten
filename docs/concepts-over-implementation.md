## Prefer Concepts over Implementation

- babel -> compilers
- webpack -> bundlers
- javascript -> programming logic
- sql -> databases
- redis -> cache
- serverless, k8s, aws -> containers, linux, vm, DNS, IP
- express -> HTTP/TCP/UDP
- react -> html, css, js
